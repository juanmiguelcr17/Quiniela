Quiniela con MVC4

El proyecto consiste en una quiniela que permita a usuarios crearlas, unirse a una existente, pronosticar los resultados de los partidos,
ver su posición en la tabla general de la quiniela. 
Se tiene contemplado el acceso por usuario y contraseña, con manejo de roles para que un usuario con rol administrador pueda modificar
información como dar de alta equipos, temporadas, ligas, etc.

Este proyecto se realizó con la intención de aprender a programar en base al patrón de diseño MVC, utilizando .Net MVC4, además de practicar
Javascript, AJAX y CSS3.
